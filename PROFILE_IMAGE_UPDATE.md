# How to Update Your Profile Image

## Quick Steps:

1. **Locate your profile image file** that you want to use (the one you uploaded)

2. **Navigate to the project folder:**
   ```
   d:\02 Code\Important Projects\Portfolio\src\assets\
   ```

3. **Replace the file:**
   - Backup the current `profile.jpg` (optional)
   - Copy your new profile image to this folder
   - Name it exactly: `profile.jpg`

4. **The image will automatically update on:**
   - Hero/Landing page (left side with animation)
   - Navigation bar (profile avatar next to logo)
   - About section (profile card)
   - Footer (small avatar)
   - Contact section (hidden, but referenced)

## Image Requirements:
- **Format:** JPG or PNG
- **Size:** At least 400x500px (for best quality)
- **Aspect Ratio:** Portrait/vertical preferred
- **File Name:** `profile.jpg`
- **Location:** `src/assets/profile.jpg`

## Tips:
- Use a professional headshot for best results
- The image will be grayscaled with a slight overlay effect on the landing page
- Ensure good lighting and contrast for luxury feel

After replacing the file, your dev server will automatically hot-reload and show the new image!
